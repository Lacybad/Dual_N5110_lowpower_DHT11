# Dual_N5110_lowpower_DHT11
Temperature and humidity on separate Nokia 5110 LCDs

# YouTube video:
https://youtu.be/a2o2LGGaHMM

# Connections:
## Nokia 5110 LCD

|No|Signal | LCD#1|LCD#2|
|-|-|-|-|
| 1|RST | 9 | 6 |
| 2|CS/CE | 10 | 5|
| 3|DC| 8 | 8|
| 4| MOSI/DIN | 11|11|
|5| SCK/CLK |  13|13|
|6| VCC  | Vcc|Vcc| 
|7| LIGHT |NC|NC|
|8| GND |GND|GND|

## DHT11 
|No|Signal|Pin|
|-|-|-|
|1|VCC|7|
|2|DATA|2|
|3|NC||
|4|GND|GND|
